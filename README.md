##  Краткое описание

В репозитории представлен Jupyter Notebook с полным циклом проверки A/A- и A/B-тестов:

1. **Подготовка данных:** загрузка и разделение на контрольную (4 четверга) и тестовую (1 четверг) группы.
2. **Визуализация:** boxplot распределений метрик и гистограммы возраста.
3. **A/A-тест ковариат:** проверка однородности по полу, ОС и возрасту (t-test Уэлча, χ², поправка Холма).
4. **A/A-тест метрик:** попарное сравнение контрольных дней с помощью бутстрапа для оценки возможности их объединения.
5. **A/B-тест:** сравнение контрольной группы (объединённой) и тестовой по CTR (бутстрап группового CTR) и по абсолютным метрикам (лайки, просмотры) с каждым контрольным днём отдельно.

Почему групповой CTR?
Персональный CTR (среднее индивидуальных CTR) даёт одинаковый вес пользователям с разным числом просмотров. Групповой CTR корректно отражает общую вовлечённость: сколько лайков приходится на каждый просмотр в целом по группе.

**Главный вывод:** новая реклама привлекла *менее активных* пользователей (меньше просмотров и лайков), но их *вовлечённость* (CTR) осталась на прежнем уровне. Выводы статистически обоснованы.

##  Используемые методы

| Задача | Статистический инструмент |
| :--- | :--- |
| Проверка однородности по возрасту | T-test Уэлча + поправка Холма |
| Проверка однородности категориальных ковариат | Критерий χ² (хи-квадрат) |
| Сравнение CTR между днями | **Бутстрап группового CTR** (sum(likes)/sum(views)) |
| Сравнение абсолютных метрик (лайки, просмотры) | **Бутстрап разности средних** |
| Множественная проверка гипотез | Метод Холма |

##  Основные выводы

1. **Ковариаты однородны:** тестовая и контрольные группы не различаются по возрасту, полу и операционной системе.

2. **Контрольные дни неоднородны по лайкам/просмотрам:** в контрольной группе наблюдается органический рост активности от четверга к четвергу, поэтому объединять их для A/B-теста по абсолютным метрикам нельзя.

3. **CTR не изменился:**  
   - Групповой CTR контрольной и тестовой групп статистически не различается (p-value > 0.05, доверительный интервал включает 0).  
   - Пользователи новой кампании ставят лайки с той же вероятностью при просмотре поста.

4. **Абсолютная активность упала:**  
   - Тестовая группа демонстрирует значимо меньше просмотров и лайков на пользователя, чем каждый из четырёх контрольных дней.

**Интерпретация:**  
Новая реклама привлекла менее активную аудиторию, но это не связано с качеством контента (CTR стабилен). Причины оттока, вероятно, лежат вне рекламной кампании (UX, технические аспекты, несоответствие ожиданий продукта).
